# Программа расчета стоимости изделий

## Описание
Программа для расчета стоимости изделий на основе технологических операций и материалов, с возможностью формирования карточек изделий, расчетов и отчетов.

## Требования
- Windows 7/8/10/11
- Установленный Python 3.7+ (для исходного кода)

## Установка и запуск

### Вариант 1: Готовый исполняемый файл
1. Скачайте архив с готовым приложением
2. Распакуйте в любую папку
3. Запустите ProductCalculator.exe

### Вариант 2: Из исходного кода
1. Установите Python 3.7+
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите: `python main.py`

## Использование

### Импорт справочника материалов
1. Перейдите в меню "Справочники" → "Импорт материалов"
2. Выберите Excel файл с материалами
3. Материалы будут загружены в систему

### Создание изделия
1. Перейдите на вкладку "Ввод данных"
2. Заполните информацию об изделии (ID, артикул, название)
3. Добавьте технологические операции
4. Добавьте материалы
5. Сохраните изделие (Файл → Сохранить)

### Просмотр каталога
1. Перейдите на вкладку "Каталог"
2. Используйте поиск для нахождения изделий
3. Выберите изделие и экспортируйте в Excel или PDF

## Функции
- Расчет стоимости операций
- Расчет стоимости материалов
- Формирование отчетов
- Экспорт в Excel с форматированием
- Экспорт в PDF
- Работа с базой данных изделий

## Структура данных
- data/database.db - основная база данных
- data/products/ - карточки изделий
